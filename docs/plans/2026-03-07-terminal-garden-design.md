# Terminal Garden - 设计文档

一个在终端中运行的植物生长视觉玩具。玩家移动光标种下种子，看植物随时间生长、开花、枯萎，屏幕如同一个小生态箱。

## 技术选型

- **语言：** Python 3
- **TUI 库：** curses（标准库，零依赖）
- **视觉风格：** ASCII 字符为主 + ANSI 颜色

## 世界模型

- 终端屏幕映射为 2D 网格（`width x height`），自动适配终端大小
- 每个格子可以为空或被一个植物的某个部分占据
- 植物对象属性：
  - `position (x, y)` — 根部位置
  - `plant_type` — 草/花/树/仙人掌
  - `age` — 经过的 tick 数
  - `stage` — seed → sprout → bloom → wilt（由 age 驱动）
  - `cells` — 植物占据的所有格子列表

## 植物类型

| 植物 | 生长速度 | 最终大小 | 视觉变化 |
|------|---------|---------|---------|
| 草 Grass | 快（15 tick） | 1x1~1x2 | `.` → `v` → `w` → `,` |
| 花 Flower | 中（25 tick） | 1x3 | `.` → `\|` → `\|/` + `*` → `;` |
| 树 Tree | 慢（50 tick） | 5x7 | `.` → `\|` → 树冠+树干 → 枯枝 |
| 仙人掌 Cactus | 慢（40 tick） | 3x5 | `.` → `\|` → 分叉形态 → 干枯 |

### 颜色方案

- seed: 白色/暗灰
- sprout: 亮绿
- bloom: 花=黄/品红, 树=深绿, 仙人掌=绿+黄花
- wilt: 棕色/暗灰

## 交互控制

| 按键 | 功能 |
|------|------|
| 方向键 / hjkl | 移动光标 |
| 1-4 | 选择植物类型 |
| Enter / Space | 种下种子 |
| p | 暂停/继续 |
| +/- | 加速/减速 |
| q | 退出 |

### 光标

反色块显示，种植时预览当前选中植物。

### 状态栏

屏幕底部：`[Grass] | Tick: 142 | Speed: 1x | Plants: 12 | [P]ause [Q]uit`

### 碰撞规则

- 不能在已有植物格子上种植
- 植物生长遇到已占据格子则该方向停止

## 架构

```
terminal-garden/
├── main.py          # 入口，curses 初始化，主循环
├── grid.py          # 网格世界，碰撞检测
├── plants.py        # 植物基类 + 4种植物定义
├── renderer.py      # 渲染引擎，颜色管理
└── config.py        # 植物配置数据（形态模板、速度等）
```

### 主循环

```
初始化 curses → 创建 Grid → 主循环:
  1. 处理输入（非阻塞）
  2. 如果未暂停：更新所有植物（age++, 阶段转换, 扩展cells）
  3. 渲染：清屏 → 画植物 → 画光标 → 画状态栏 → refresh
  4. halfdelay 控制帧率
```

### 生长算法

每种植物定义一组"形态模板"（按阶段），模板是相对于根部位置的字符偏移列表。进入新阶段时切换模板，新 cells 注册到 grid（检查碰撞）。
